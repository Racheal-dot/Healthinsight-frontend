About HealthInsight

HealthInsight is an AI-powered health assessment web application developed as a final year project to support preliminary health evaluation and provide intelligent recommendations based on user health information.

Case Study Institution

This project was developed using the Pogil College of Health Technology School Clinic as the case study institution.

The system is designed to support students and staff by providing preliminary health assessments, personalized health advice, and specialist recommendations.

It aims to improve access to health information and encourage early consultation at the school clinic.

Project Overview

The application collects basic health data such as age, gender, weight, height, blood pressure, symptoms, and lifestyle information. It then sends the data to an intelligent diagnostic engine powered by Infermedica's medical API.

Based on the submitted information, the system predicts possible medical conditions, estimates their probability, and provides personalized health advice to help users make informed decisions about their wellbeing.

Objectives of the Project

  • To develop a web-based health diagnostic system.
  • To integrate Artificial Intelligence into healthcare support.
  • To predict likely medical conditions from user symptoms.
  • To provide personalized health recommendations.
  • To improve awareness of preventive healthcare practices.

Key Features

  • User-friendly health assessment form.
  • Automated BMI calculation.
  • AI-based disease prediction using Infermedica.
  • Risk level classification (Low, Medium, High).
  • Personalized health advice.
  • Printable diagnostic report.
  • Responsive design for desktop and mobile devices.

Technologies Used

  • HTML5 - Structure of web pages.
  • CSS3 - Styling and responsive design.
  • JavaScript - Dynamic functionality and API communication.
  • PHP - Backend processing.
  • Infermedica API - Medical diagnosis engine.
  • Railway - Backend hosting.
  • Netlify - Frontend hosting.
  • GitHub - Version control and deployment.

How the System Works

  1. The user enters personal and symptom data.
  2. The frontend validates the information.
  3. Data is sent to the PHP backend hosted on Railway.
  4. The backend forwards the request to Infermedica.
  5. Infermedica returns possible diagnoses.
  6. The application displays results and health advice.

Benefits of HealthInsight

  • Provides instant preliminary health insights.
  • Encourages proactive health monitoring.
  • Offers personalized health recommendations.
  • Supports educational use in healthcare informatics.
  • Accessible from any internet-enabled device.

Project Scope

HealthInsight is designed for educational and research purposes. It is not intended to replace licensed medical professionals or formal clinical diagnosis.
